# react-mobx-admin-panel

<img src="https://i.imgur.com/KKc5TOa.png"/>

Scallable and easy to use admin dashboard that will help you to manage your REST API data.

## DEMO
https://rest-admin-panel.herokuapp.com/

## Installation

Use these steps to install project

```
1. yarn install
2. yarn start
```

Build project:

```
yarn build
```

## Usage

- Uses webpack to build project.
- Uses Babel to compile ES6 and ES7 code.
- You can use SASS\SCSS within this boilerplate.
- Check your code with ESLint and Prettier.
- Supports styled components.
- Supports hot-module-reload.
